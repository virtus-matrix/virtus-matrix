# Virtus Matrix - Landing Page

A premium, modern SaaS-style responsive landing page for Virtus Matrix, an academic and digital solutions provider.

## 🚀 Tech Stack
- **HTML5 & CSS3**
- **JavaScript (Vanilla)**
- **Bootstrap 5** (Layout & Components)
- **Font Awesome 6** (Icons)
- **AOS** (Scroll Animations)
